TrainForge

SaaS para personal trainers e alunos com desafios semanais, ranking global, histórico de performance, gestão de serviços e painel administrativo.

Front-end React, React Router, Axios, Recharts, Framer Motion, Formik + Yup, Bootstrap/Tailwind.
Back-end Node.js + Express, JWT com refresh rotativo, rate limit, RBAC (admin/trainer/client).
Dados MySQL (WAMP), historico de treino, desafios, servicos, tickets de suporte e feedback.
