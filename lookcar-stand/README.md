# LookCar Stand (Protótipo)

Landing moderna para stand automotivo, inspirada na Lookcar: hero com filtro rápido, stock filtrável, diferenciais, testimoniais e CTA.

## Stack
- HTML + CSS (theme escuro com gradientes e glassy)
- JavaScript simples para filtros (busca, tag, tipo e orçamento)
- Sem build, pronto para servir estático

## Estrutura
- `index.html` — layout principal
- `style.css` — tema, grelhas, botões e cartões
- `script.js` — dados mock de viaturas e filtros
- (opcional) adicionar imagens reais em `assets/img/`

## Próximos passos
- Integrar APIs reais de stock e financiamento
- Adicionar páginas de detalhe de viatura e formulário de lead
- SEO (metadados, schema de produto) e analytics

## Executar
Servir a pasta `projects/lookcar-stand/` e abrir `index.html` no browser.
