# Avaliacao e Testes

## Estrategia
1. Testes de unidade.
2. Testes de integracao (app, servidor, dispositivos, protocolos).
3. Testes de interface com utilizadores.
4. Testes de desempenho sob carga.
5. Testes de seguranca e intrusao.
6. Testes de aceitacao do utilizador.

## Metricas
- Usabilidade: tempo de configuracao, passos por tarefa.
- Desempenho: tempo de resposta (<1s), disponibilidade >99%.
- Seguranca: tentativas bloqueadas, dispositivos com criptografia.
- Satisfacao: NPS, taxa de retencao.
- Suporte: tempo medio de resolucao, taxa de suporte ativada.
